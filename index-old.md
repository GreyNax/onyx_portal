# Building in Public

This is a live documentation of developing the Onyx Portal - an open standard for modular, portable spaces that dock into standardized infrastructure.

**The Vision:** Start with entertainment (escape rooms, pop-up venues), scale to commerce (food service, retail), and ultimately enable affordable portable housing.

**Why Open?** This project works best as a commons. The more people who adopt the standard, the more valuable it becomes for everyone. Network effects, not proprietary control, are the goal.

## ğŸ”‘ Core Concept

### The Container
- 20' shipping container (160 sq ft interior)
- Front: Full-width doors (dramatic entrance)
- Rear: Utility connections (RV-standard plugs)
- Interior: Modular panels on French cleats (swap themes easily)

### The Hub
- Central building (lobby, restrooms, amenities)
- Radial or linear dock positions with utility blocks
- Containers back in, connect utilities, open to building
- Can swap containers in under an hour

### The Network
- Multiple hubs in different cities
- Vendors can relocate containers between hubs
- Take your business (or home) when you move
- Open standard enables competition and growth

## ğŸŒ Why Open Standard?

**Like shipping containers, USB ports, or electrical outlets** - the value is in universal compatibility, not proprietary control.

Anyone can:
- Build containers to the spec
- Operate hub locations
- Manufacture components
- Compete and innovate

**The network effect is the moat** - more hubs = more value for everyone.
