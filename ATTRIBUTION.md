This site uses the [Long Haul](https://github.com/brianmaierjr/long-haul) Jekyll theme by Brian Maier Jr.

**License:** MIT License  
**Repository:** https://github.com/brianmaierjr/long-haul  
**Author:** Brian Maier Jr. ([@brianmaierjr](https://twitter.com/brianmaierjr))

The Long Haul theme is a minimal, type-focused Jekyll theme built with SASS. It has been customized for the Obsidian Portal project while maintaining the core design and functionality.

### Theme Features Used

- Minimal, type-focused design
- SASS-based styling
- Responsive navigation
- Dark mode support
- Pagination
- SVG social icons

## Modifications

The theme has been adapted for this project with the following changes:
- Custom navigation structure for project-specific pages
- Additional page layouts for collections (research, decisions)
- Updated footer attribution
- Removed template-specific navigation links
- Customized configuration for project needs
